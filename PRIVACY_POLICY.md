# PDFxcel 개인정보 처리방침

**최종 업데이트:** 2025년 7월 28일  
**시행일:** 2025년 7월 28일

## 1. 개인정보 처리방침 개요

PDFxcel("우리", "당사")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보 처리방침은 PDFxcel 모바일 애플리케이션(이하 "앱")을 사용할 때 수집, 사용, 보관되는 정보에 대해 설명합니다.

## 2. 수집하는 정보

### 2.1 자동으로 수집되는 정보
- **디바이스 정보**: 운영체제 버전, 디바이스 모델, 앱 버전
- **사용 통계**: 앱 사용 빈도, 변환 완료 횟수 (개인 식별 불가)
- **오류 로그**: 앱 안정성 개선을 위한 익명화된 오류 정보

### 2.2 사용자가 제공하는 정보
- **PDF 파일**: 변환을 위해 업로드하는 PDF 문서
- **사용 설정**: AI 변환 사용 여부 등 앱 내 설정

### 2.3 수집하지 않는 정보
- **개인 식별 정보**: 이름, 이메일, 전화번호 등
- **위치 정보**: GPS 좌표, 위치 데이터
- **연락처**: 주소록, 통화 기록
- **사진/미디어**: PDF 파일 외의 개인 사진이나 미디어 파일

## 3. 정보 사용 목적

### 3.1 핵심 서비스 제공
- PDF 파일을 Excel 형식으로 변환
- AI 기반 데이터 분석 및 추출
- 변환된 파일 다운로드 서비스

### 3.2 서비스 개선
- 앱 성능 최적화
- 변환 정확도 향상
- 사용자 경험 개선

### 3.3 기술적 운영
- 오류 수정 및 버그 해결
- 서비스 안정성 유지
- 보안 강화

## 4. 정보 보관 및 삭제

### 4.1 PDF 파일 처리
- **업로드된 PDF**: 변환 완료 후 **24시간 이내 자동 삭제**
- **변환된 Excel**: 다운로드 후 **7일 이내 자동 삭제**
- **임시 파일**: 변환 과정에서 생성되는 임시 파일은 **즉시 삭제**

### 4.2 사용 데이터
- 익명화된 사용 통계는 서비스 개선 목적으로만 보관
- 개인 식별이 불가능한 형태로만 유지

## 5. 정보 공유 및 제3자 제공

### 5.1 제3자 제공 금지
- **개인정보는 절대 제3자에게 판매, 임대, 제공하지 않습니다**
- 사용자의 명시적 동의 없이 개인정보를 공유하지 않습니다

### 5.2 예외적 상황
다음의 경우에만 법적 요구에 따라 정보를 제공할 수 있습니다:
- 법원의 명령 또는 법적 절차
- 긴급한 생명 위험 상황
- 서비스 약관 위반 조사

### 5.3 사용하는 외부 서비스
- **Railway.app**: 서버 호스팅 (PDF 파일 임시 저장)
- **Anthropic Claude API**: AI 기반 데이터 추출
- **Expo**: 앱 빌드 및 배포 플랫폼

## 6. 데이터 보안

### 6.1 기술적 보안조치
- **HTTPS 암호화**: 모든 데이터 전송 시 SSL/TLS 암호화
- **서버 보안**: 최신 보안 패치 적용
- **접근 제한**: 개발자만 서버 접근 가능

### 6.2 운영적 보안조치
- **자동 삭제**: 설정된 기간 후 자동 파일 삭제
- **로그 최소화**: 필요한 정보만 최소한으로 기록
- **정기 점검**: 보안 취약점 정기 점검

## 9. 개인정보 처리방침 변경

### 9.1 변경 통지
- 개인정보 처리방침이 변경되는 경우 앱 내 공지사항을 통해 알려드립니다
- 중요한 변경사항은 업데이트 최소 7일 전에 사전 공지합니다

### 9.2 지속적인 사용
변경된 개인정보 처리방침 공지 후 앱을 계속 사용하시면 변경사항에 동의한 것으로 간주됩니다.

## 10. 관련 법규 준수

PDFxcel은 다음 법규를 준수합니다:
- **개인정보보호법** (대한민국)
- **정보통신망 이용촉진 및 정보보호 등에 관한 법률**
- **Apple App Store 개인정보 보호 정책**

## 11. 국제 사용자

PDFxcel은 전 세계 사용자를 대상으로 서비스를 제공합니다. 사용자의 데이터는 서비스 제공을 위해 필요한 국가의 서버에 저장될 수 있으며, 해당 국가의 데이터 보호 법규가 적용됩니다.
