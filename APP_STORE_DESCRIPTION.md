# PDFxcel - App Store 설명

## 앱 제목
PDFxcel - PDF to Excel 변환기

## 부제목
AI로 은행명세서를 Excel로 스마트 변환

## 키워드
PDF, Excel, 변환, 은행명세서, AI, 자동화, 문서관리, 스프레드시트

## 앱 설명

### 📋 앱 소개
PDFxcel은 PDF 형태의 은행 명세서를 Excel 파일로 쉽고 정확하게 변환해주는 AI 기반 모바일 앱입니다. 복잡한 표 구조도 Claude AI 기술로 정밀하게 분석하여 완벽한 Excel 스프레드시트로 변환합니다.

### ✨ 주요 기능
• **간편한 업로드**: PDF 파일을 선택하여 쉽게 업로드
• **AI 지능형 분석**: Claude AI로 정확한 데이터 추출 및 구조화
• **Excel 변환**: 구조화된 Excel 파일로 자동 변환
• **즉시 다운로드**: 변환 완료 즉시 Excel 파일 다운로드
• **쉬운 공유**: 변환된 파일을 바로 공유하거나 저장

### 🎯 사용법
1. PDF 파일 선택 (은행 명세서)
2. AI 분석 모드 선택
3. 변환 시작 버튼 클릭
4. Excel 파일 다운로드 및 공유

### 🔒 보안 및 개인정보
• 업로드된 파일은 24시간 후 자동 삭제
• 모든 데이터는 HTTPS로 암호화 전송
• 개인정보는 절대 제3자에게 제공하지 않음
• 개인정보 처리방침 완전 준수

### 💰 요금제
• **무료**: 월 3회 기본 변환
• **PRO**: 무제한 AI 변환 (추후 출시)

### 📱 지원 환경
• iOS 13.0 이상
• iPhone, iPad 지원
• 오프라인 사용 불가 (변환 처리를 위해 인터넷 연결 필요)

### 🎨 특징
• 직관적이고 깔끔한 사용자 인터페이스
• 빠른 변환 속도
• 높은 정확도의 데이터 추출
• 실시간 진행상황 표시
