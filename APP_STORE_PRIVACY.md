# PDFxcel - App Store 개인정보 처리 정책

## 수집하는 데이터

### 📄 사용자 콘텐츠
- **PDF 파일**: 변환 서비스 제공을 위해 임시 수집
- **변환 설정**: 사용자가 선택한 변환 옵션

### 📊 사용 데이터  
- **앱 사용 통계**: 익명화된 사용 빈도 및 기능 사용 현황
- **성능 데이터**: 앱 성능 개선을 위한 익명 데이터

## 데이터 사용 목적

✅ **PDF to Excel 변환 서비스 제공**  
✅ **AI 기반 데이터 추출 및 분석**  
✅ **앱 성능 및 사용자 경험 개선**  
✅ **기술적 문제 해결 및 버그 수정**

## 데이터 보호

🔒 **자동 삭제**: 업로드된 PDF는 24시간 내 자동 삭제  
🔒 **암호화 전송**: 모든 데이터는 HTTPS로 암호화 전송  
🔒 **제3자 미공유**: 개인 데이터는 절대 제3자에게 제공하지 않음  
🔒 **최소 수집**: 서비스 제공에 필요한 최소한의 데이터만 수집

## 제3자 서비스

- **Railway.app**: 서버 호스팅
- **Anthropic Claude**: AI 데이터 분석
- **Expo**: 앱 플랫폼 서비스

이들 서비스는 각각의 개인정보 처리방침을 따릅니다.
